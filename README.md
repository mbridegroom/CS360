# CS360

1. Briefly summarize the requirements and goals of the app you developed. What user needs was this app designed to address?

  The app was developed to help users securely create, organize, and manage personal events efficiently. It was designed to address the need for an easy-to-use event management system that keeps track of important dates.

2. What screens and features were necessary to support user needs and produce a user-centered UI for the app? How did your UI designs keep users in mind? Why were your designs successful?

  It was designed to address the need for an easy-to-use event management system that keeps track of important dates. Necessary screens included login, event list, and event management screens, with features for adding, updating, and deleting events.

3. How did you approach the process of coding your app? What techniques or strategies did you use? How could those techniques or strategies be applied in the future?

  The UI was designed with simplicity, clear labels, and intuitive navigation to ensure users could easily interact with the app. I used modular code, reusable components, and SQLite for data storage, focusing on clean architecture and maintainable code.

4. How did you test to ensure your code was functional? Why is this process important, and what did it reveal?

  These strategies can be applied to future apps to streamline development and reduce debugging time. I tested each function using sample data and real interactions to ensure proper functionality and reliability.

5. Consider the full app design and development process from initial planning to finalization. Where did you have to innovate to overcome a challenge?

  This process revealed errors early and ensured that event management features worked as expected for all users.

6. In what specific component of your mobile app were you particularly successful in demonstrating your knowledge, skills, and experience?

  I had to innovate when integrating the database with the UI to handle dynamic event updates without crashing the app.
